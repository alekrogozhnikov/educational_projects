**Задача**

Проанализировать и дать рекомендацию о том, на каких играх стоит сосредоточиться в следующем году, чтоб выстроить рекламную компанию.

**Описание проекта**

Выявлены параметры, определяющие успешность игры в разных регионах мира. На основании этого подготовлен отчет для магазина компьютерных игр для планирования рекламных кампаний. Составлены портреты пользователей каждого региона. Балв выполнена проверка гипотез с помощью критерия Стьюдента для независимых выборок.
